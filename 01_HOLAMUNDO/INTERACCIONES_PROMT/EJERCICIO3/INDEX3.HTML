EJERCICIO 3

Crea una variable llamada nombreUsuario y otra llamada contraseña que almacene un nombre de usuario y una contraseña, respectivamente. Escribe un programa en JavaScript que verifique si el nombre de usuario es "usuario123" y la contraseña es "secreto". Si ambos son correctos, imprime "Acceso concedido"; de lo contrario, imprime "Acceso denegado". Utiliza una declaración "if" para realizar la verificación.
